# Mô tả dự án
-Dự án mô tả việc đọc giá trị quang trở thông quả bộ ADC của esp32.
-Đọc giá trị từ cảm biến ánh sáng (LDR) và hiển thị trên máy tính dưới dạng đồ thị (plot) thông qua giao tiếp UART.
